# Project Startup

## Prerequisites

- Node.js version: **v24.2.0**

## Setup

1. Create a `.env` file in the root directory of the project and add the following variables:

   ```env
   VITE_MAPS_API_KEY=your_key
   VITE_MAP_ID=mock_map_id
   ```

## Install dependencies and run

```bash
npm install
npm run dev
```
